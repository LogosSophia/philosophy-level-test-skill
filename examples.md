# 示例 / Examples

以下示例用于说明 skill 的输出风格。实际评分可以根据上下文微调，但应保持严格。

These examples show the intended output style. Actual scores may vary with context, but the scoring should remain strict.

---

## 示例 1：自由 = 无限制

### 输入

```text
我认为自由就是摆脱一切限制。真正自由的人不应该被任何规则束缚。
```

### 输出示例

```text
### 哲学水平评分：48/100

### 段位：有问题意识的思考者

### 伪深刻指数：55/100

### 核心命题
自由被理解为完全摆脱限制，因此真正自由的人不应受规则束缚。

### 核心概念
自由、限制、规则、真正自由的人

### 一句话判定
这段话有一个真问题，但把心理自由、存在论自由和社会规则问题混在了一起，概念还没有站稳。

### 最强之处
它触及了自由与限制之间的关系，这是一个真正的哲学问题。

### 最大漏洞
概念偷换、尺度漂移、结论冒充前提。

### 具体审查
- 对象资格：自由没有被切清楚，不知道是心理状态、政治权利，还是存在论状态。
- 说话位置：文本直接裁决“真正自由的人”，但没有说明判断标准从哪里来。
- 概念稳定性：前半句的自由偏心理/存在论，后半句变成社会规则问题。
- 尺度控制：从“摆脱限制”直接跳到“不应受规则束缚”，缺少中介。
- 非法综合风险：较低，但有把“无限制”直接等同于“自由”的定义压缩问题。
- 论证强度：主要是宣言，推理链较短。

### 致命追问
1. 如果自由是完全没有限制，那么它如何区别于虚无？
2. 规则一定是自由的敌人，还是也可能是自由的条件？
3. 你说的自由是心理自由、政治自由，还是存在论自由？

### 修改建议
先把“自由”缩小到一个明确层级。不要一开始谈“一切限制”，可以先讨论“心理自由是否需要边界”。
```

## 示例 2：爱 = 定义项之和

### 输入

```text
爱就是关心、承诺、欲望和责任的综合。只要这些要素都在，就能说明爱是什么。
```

### 输出示例

```text
### 哲学水平评分：42/100

### 段位：有问题意识的思考者

### 伪深刻指数：62/100

### 核心命题
爱可以通过关心、承诺、欲望和责任等要素的综合来定义。

### 核心概念
爱、关心、承诺、欲望、责任、综合

### 一句话判定
这段话最大的问题不是词不清，而是发生了典型的信息丢失型非法综合。

### 最强之处
它试图分析“爱”的构成条件，说明文本至少有概念拆分意识。

### 最大漏洞
非法综合、定义冒充原物、条件列表冒充生成机制。

### 具体审查
- 对象资格：“爱”被当成可定义对象，但没有说明爱是情感、关系、行动，还是持续发生。
- 说话位置：问题不严重，文本主要在概念定义层说话。
- 概念稳定性：关心、承诺、欲望和责任各自含义尚未说明。
- 尺度控制：尺度漂移不明显。
- 非法综合风险：很高。关心、承诺、欲望、责任只是对爱的切片，不能加起来自动还原爱本身。
- 论证强度：列出要素，但没有说明这些要素如何生成爱。

### 致命追问
1. 关心、承诺、欲望和责任都存在时，为什么一定生成爱，而不是义务、占有或习惯？
2. 爱作为发生被定义成名词切片之后，丢失了哪些信息？
3. 这些要素之间的生成机制是什么？

### 修改建议
不要说“爱就是这些要素的综合”。可以改成：“关心、承诺、欲望和责任可能是爱发生后的可观察切面，但还不能解释爱如何生成。”
```

## 示例 3：历史终局

### 输入

```text
人类历史最终会走向精神统一，文明的一切冲突都会在更高的意识中被解决。
```

### 输出示例

```text
### 哲学水平评分：35/100

### 段位：哲学气氛组

### 伪深刻指数：78/100

### 核心命题
人类历史有一个精神统一的终局，文明冲突会在更高意识中被解决。

### 核心概念
人类历史、精神统一、文明冲突、更高意识、解决

### 一句话判定
这段话哲学气氛很足，但位置、对象、尺度和综合关系都没有交代，主要靠宏大词汇推进。

### 最强之处
它触及历史整体、文明冲突与精神整合的问题，方向本身有哲学潜力。

### 最大漏洞
位置非法、伪整体、尺度漂移、终极拼图型非法综合。

### 具体审查
- 对象资格：“人类历史”“精神统一”“更高意识”都没有清楚边界。
- 说话位置：文本站在历史之内，却声称看到了历史最终方向。
- 概念稳定性：“精神”和“意识”可能被混用。
- 尺度控制：从文明冲突直接跳到更高意识，缺少承载机制。
- 非法综合风险：高。历史、文明、意识被拼成终极闭环，但每一环的转换机制没有说明。
- 论证强度：基本是宏大宣言，没有实际论证路径。

### 致命追问
1. 你凭什么站在历史之内判断历史最终会怎样？
2. “更高意识”是什么对象？它在哪里承载文明冲突？
3. 冲突为什么一定会被统一，而不是持续分化、沉积或转移？

### 修改建议
先撤掉“最终”这个词。可以从一个更小的问题开始：文明冲突在什么条件下会从暴力冲突转化为制度性协调？
```

## 示例 4：语言边界

### 输入

```text
如果我们永远在语言之内，那么我们凭什么说语言之外有什么？
```

### 输出示例

```text
### 哲学水平评分：78/100

### 段位：前提层玩家

### 伪深刻指数：22/100

### 核心命题
如果主体无法离开语言，那么关于语言之外的判断就需要重新审查其合法性。

### 核心概念
语言之内、语言之外、凭什么说、判断合法性

### 一句话判定
这是一个真正的前提层问题，重点不是给出答案，而是审查说话位置是否成立。

### 最强之处
它没有直接断言“语言之外不存在”，而是追问我们是否有资格谈论语言之外，这一点很强。

### 最大漏洞
对象边界仍需补充。

### 具体审查
- 对象资格：“语言之内/之外”需要进一步说明，但问题本身已经意识到边界问题。
- 说话位置：文本主动审查说话位置，这是加分项。
- 概念稳定性：语言概念相对稳定，但仍需区分自然语言、符号系统、意义结构。
- 尺度控制：尺度跳跃不严重。
- 非法综合风险：低。
- 论证强度：以追问形式成立，结构清楚。

### 致命追问
1. “语言之内”是指自然语言之内，还是意义结构之内？
2. 说“语言之外”时，这个“之外”是否已经被语言标记过？
3. 如果无法离开语言，我们还能否区分语言边界与语言内部差异？

### 修改建议
下一步应定义“语言之内”的层级：是表达工具、意义结构、主体条件，还是世界边界。
```
