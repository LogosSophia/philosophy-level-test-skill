# 术语表 / Glossary

本术语表用于稳定 skill 的中英表达，避免同一概念被 AI 随意换译。

This glossary stabilizes Chinese-English terminology so that the AI does not randomly translate the same concept in different ways.

| 中文术语 | English Term | 一句话定义 |
|---|---|---|
| 哲学水平测试 | Philosophy Level Test | 对一段哲学思考进行结构强度、概念成本和伪深刻风险评测。 |
| 概念成本 | Conceptual Cost | 使用一个概念时必须承担的定义、边界、层级、位置和论证负担。 |
| 对象资格 | Object Qualification | 一个词是否有资格被当成对象来讨论。 |
| 对象未成立 | Unqualified Object | 把一个词当成对象讨论，却没有说明它是什么、边界在哪里、如何被切出。 |
| 说话位置 | Speaking Position | 文本中的判断者站在什么层级、边界和账本中说话。 |
| 位置非法 | Invalid Position | 站在边界之内，却声称自己把握到了边界本身和边界之外。 |
| 概念偷换 | Concept Substitution | 同一个词前后换了意思，但作者没有承认转换。 |
| 尺度漂移 | Scale Drift | 从小范围判断突然跳到大范围结论，中间没有说明放大机制。 |
| 非法综合 | Illegal Synthesis | 把被切分、定义、抽象、记录或跨尺度转译后的碎片，拼成据称等同于原对象、原发生或整体的东西。 |
| 信息丢失 | Information Loss | 动词性发生被定义、名词化或记录化时，原发生的部分结构已经丢失。 |
| 动词性发生 | Verbal Occurrence | 知、爱、说、变、切、指、觉、生成等正在发生的动作或过程。 |
| 名词切片 | Nominal Slice | 对发生或整体进行定义、分类、抽象之后得到的静态概念片段。 |
| 记录冒充发生 | Record Mistaken for Occurrence | 把发生之后留下的痕迹、描述、数据、定义、文本或记忆当成发生本身。 |
| 条件列表冒充生成机制 | Condition List Mistaken for Generative Mechanism | 列出一堆必要条件后，直接声称已经解释了对象如何生成。 |
| 层级拼接 | Layer Splicing | 把不同账本、不同层级、不同对象资格的东西直接拼成一个统一对象。 |
| 后设分析冒充原初根据 | Retrospective Analysis Mistaken for Original Ground | 把后来分析出来的结构，倒置成对象最初成立的原因。 |
| 伪整体 | Pseudo-Whole | 把无法合法看见或切出的整体，说成一个已经被把握的完整对象。 |
| 隐喻冒充本体 | Metaphor Mistaken for Ontology | 本来只是比喻、诗性表达或情绪图像，却被当成真实结构来论证。 |
| 情绪代替论证 | Emotion Replacing Argument | 用强烈感受、姿态或宣言代替理由。 |
| 结论冒充前提 | Conclusion Posing as Premise | 把本来需要证明的结论，提前当成推理起点使用。 |
| 矛盾包装 | Contradiction Packaging | 概念互相冲突时，不解决冲突，而把冲突说成更高层次的深刻。 |
| 名词堆砌 | Nominal Stacking | 用一串高级名词制造深度感，但这些名词没有真正参与推理。 |
| 伪深刻 | Pseudo-Depth | 语言看似深奥，但对象、位置、论证和概念成本没有承担起来。 |
| 真问题意识 | Genuine Problem Awareness | 即使表达不成熟，也确实碰到了值得追问的前提、对象、主体、语言或存在问题。 |
| 致命追问 | Fatal Question | 能真正卡住文本，使其必须补充前提、边界、位置或机制的问题。 |

## 翻译固定建议 / Translation Notes

- “非法综合”固定翻译为 **Illegal Synthesis**。
- “位置非法”固定翻译为 **Invalid Position**。
- “对象未成立”固定翻译为 **Unqualified Object**。
- “伪深刻”固定翻译为 **Pseudo-Depth**。
- “伪整体”固定翻译为 **Pseudo-Whole**。
- “尺度漂移”固定翻译为 **Scale Drift**。

不要在同一个文档中把同一术语随意改译为其他表达。除非有特殊理由，不要把 Illegal Synthesis 改成 invalid integration、unlawful composition 或 false synthesis。
