# 更新日志 / Changelog

本项目遵循语义化版本风格，但早期版本以内容迭代为主。

---

## v0.1.0 - 2026-06-24

初始公开版本。

新增：

- `README.md`：中英双语介绍。
- `philosophy_level_test.skill.md`：核心 skill 规则。
- `scoring.md`：100 分评分说明。
- `glossary.md`：中英术语表。
- `examples.md`：示例输入与输出。
- `QUICKSTART.md`：快速开始指南。
- `prompt_templates.md`：多场景 prompt 模板。
- `CONTRIBUTING.md`：贡献指南。
- `ROADMAP.md`：后续路线图。

核心审查项：

- 对象未成立
- 位置非法
- 概念偷换
- 尺度漂移
- 非法综合
- 伪整体
- 隐喻冒充本体
- 情绪代替论证
- 结论冒充前提
- 矛盾包装
- 名词堆砌
- 伪深刻
- 真问题意识

特别说明：

本版本将“非法综合”提升为核心审查项，不再只把它理解为名词堆砌。非法综合的核心判准是：

> 切分时已经发生信息丢失，后面再把切片加起来，也不能自动恢复原来的发生。
